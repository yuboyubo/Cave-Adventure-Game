# Cave-Adventure-Game

Run main.py to start the game

| Input | Action |
| ------- | --- |
| `-DIRECTION-' | Direction to move in |
| `I -DIRECTION-' | `I W' investigates the Entity in the given direction |
| `H' | Help text|
| `Q' | Quit |

| Input | Direction |
| ------- | --- |
| `W' | Up |
| `S' | Down |
| `A' | Left |
| `D' | Right |
